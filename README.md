# bobble-head : 

**BobbleHead for GTA SA/SAMP Android (MoonLoader/Monetloader)**

Port of ThirteenAG's BobbleHead plugin from PC to GTA SA/SAMP Android using MoonLoader/Monetloader Lua + FFI. Scales the player's head bones via RpHAnimHierarchy matrix manipulation, with an ImGui menu to toggle and adjust head size.

**Requirements :**
- MoonLoader/Monetloader
- mimgui
- SAMemory
- monethook

**Usage**

/bobblehead — open menu
